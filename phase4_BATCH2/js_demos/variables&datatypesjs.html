JS Variables

Anything in JS can be declared using var, let(from ES 2015) and const(from ES 2015)

var
-var is global to js, meaning any variable we initialize using var it remains throughout the js page and across the js pages too
let
-let is local to anything
const
-const is same as let wrt visibility but you cannot modify the value of const
