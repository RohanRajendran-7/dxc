Demo of Different Types of List

There are 3 types of List

1)Ordered List(which is ordered by some number,roman or alpha)

2)UnOrdered List(which is not ordered and is been represented with dots, circles, squares arrows)

3)Desciption List(where you have a menu item and below that you have short description about that menu item)
