In JS we can decalre any variable using var and from ES5 we got let and const as well to provide more visibility to the local scope

var
- is global scope, meaning you can access that value not only in that js but also across the files as well
let from ES5
- let is a local scope visibilty will be within the current js file itself
const from ES5
- const is same as let but it is constant i.e you cannot modify the value of it
